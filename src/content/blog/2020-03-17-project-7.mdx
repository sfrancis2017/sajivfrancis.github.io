Social media is an internet-based form of communication. Social media platforms allow users to have conversations, share information, and create web content. **University Place** is a social media web app for students for communication with friends. There are two different platforms to access this application — one used by students (Users) and another by Admins. They both have different functionalities. Each university has its own web pages, which are not interlinked. Students from the same university can chat and share their photos, project ideas, and videos.
